What is Season Pass?

Season Pass brings the excitement of battle pass progression systems to Minecraft. Players complete missions to earn XP and levels, designed specifically for multiplayer servers with performance and customization in mind.

Battle Pass Style

Familiar progression system that players love from other games

Multiplayer Ready

High Performance

How it works

A simple, intuitive progression system that keeps players engaged

1

Receive Missions

Players receive daily, weekly and seasonal missions automatically

2

Track Progress

Progress is tracked automatically for kills, mining, travel, and custom objectives

3

Complete & Earn XP

Completing missions grants XP based on difficulty and configuration

4

Level Up

XP increases your Season Pass level throughout the season

5

Unlock Rewards

Each level unlocks rewards defined by your server owner

Mission Types

Three tiers of missions to keep players engaged throughout the season

DAILY

Daily Missions

Short objectives that reset every day. Perfect for quick engagement and daily player retention.

WEEKLY

Weekly Missions

Medium-length challenges with higher rewards. Encourages sustained engagement throughout the week.

SEASONAL

Seasonal Missions

Long-term objectives lasting the entire season. Major milestones with epic rewards.

Customization & Configuration

All missions, rewards and XP values are fully configurable through separate JSON config files. Each mission type (daily, weekly, seasonal) has its own dedicated configuration file, giving server owners complete control over their progression system with easy-to-manage mission definitions.

Separate Config Files

Three dedicated JSON files: daily.json, weekly.json, and seasonal.json

Mission Types Support

Kill, break, place, travel, craft, fish, smelt, eat, sleep, anvil, and open container missions

Advanced Filters

Filter by specific entities, blocks, block tags, or entity types

XP Rewards

Configure XP values for each mission independently

Localization Keys

Mission titles and descriptions support multiple languages via translation files
