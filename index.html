Documentation & Guides

Learn how to install, configure, and get the most out of my mods.

Getting Started

All mods require Fabric to be installed first. Make sure you have the correct version for your Minecraft installation.

Fabric Installation

  1. Download the Fabric Installer
  2. Run installer and select your Minecraft version
  3. Install Fabric API mod
  4. Place mod .jar file in your mods folder

Some mods may require additional libraries or dependencies. Always check the individual mod page for specific requirements before installation.

Configuration

Most mods include configuration files that can be edited to customize your experience. These are typically found in:

config/modname.json

Example configuration for GUI Shop:

{
  "shops": [
    {
      "shopName": "Shop number one",
      "items": [
        {
          "name": "The boat",
          "material": "minecraft:acacia_chest_boat",
          "description": [
            "This is a nice boat",
            "Very beautiful"
          ],
          "buyPrice": 50.0,
          "sellPrice": 25.0,
          "nbt": "{}",
          "quantityList": [
            1
          ]
        },
        {
          "name": "BBQ Sword",
          "material": "minecraft:diamond_sword",
          "description": [],
          "buyPrice": 0.0,
          "sellPrice": 0.0,
          "nbt": "{Damage:0,Enchantments:[{id:\"fire_aspect\",lvl:2},{id:\"sweeping\",lvl:2}],display:{Lore:['[{\"text\":\"Crispy and tasty\",\"italic\":false}]'],Name:'[{\"text\":\"The BBQ\",\"italic\":false}]'}}",
          "quantityList": [

      "shopName": "A second shop",

}

Changes to config files require a game restart to take effect. Always back up your world before making significant configuration changes.

Frequently Asked Questions

Common reasons include:

  • Incorrect Minecraft version (check mod requirements)
  • Missing dependencies (check mod description)
  • Conflicts with other mods (try testing with only this mod)
  • Outdated mod loader (update Fabric)
  • Corrupted download (try re-downloading the mod)

Check your latest.log file in the logs folder for specific error messages that can help diagnose the issue.

Please report bugs on the mod's GitHub issue tracker or through the contact form below with the following information:

  • Minecraft version (e.g., 1.19.2)
  • Mod version (found in mods menu)
  • Detailed steps to reproduce the issue
  • Crash log (if applicable) from crash-reports folder
  • Screenshots or video (if helpful for visualization)
  • List of other installed mods (in case of conflicts)

The more information you provide, the faster I can identify and fix the issue!

All free mods can be used in modpacks with proper attribution.
Requirements:

  • Include a link to the original mod page
  • Clearly credit "UnsafeDodo" as the author
  • Do not redistribute the mod files directly - link to official sources
  • Do not claim the mod as your own creation

Premium mods require explicit permission for inclusion in modpacks. Please contact us for licensing information.
